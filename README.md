基于Gank.IO提供的API的第三方客户端，风格上是基于Material Design的，项目中覆盖了MVP、RxJava、Retrofit、第三方登录以及很多第三方库的使用。App可以及时的查看gank.io的最新更新，还有可以很方便的对信息进行分类查看，可以对感觉有兴趣的信息进行在线收藏，重点是里面还有很多的福利图哦。
